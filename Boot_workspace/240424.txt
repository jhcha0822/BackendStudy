REST(표현적 상태 전송) Respresentational State Transfer
서버와 클라이언트가 데이터를 주고 받을 때 데이터 표현 방법에 대한 설계(아키텍쳐)
데이터는 명사, HTTP method 요청은 동사
Restful: 로이필딩의 URL 표기방법을 준수한 표현

목록: /board GET
등록: /board POST
1건: /board/23 GET
수정: /board PUT
삭제: /board DELETE

mysql> create table board(
    ->  board_idx int primary key auto_increment
    ->  , title varchar(100)
    ->  , writer varchar(30)
    ->  , content text
    ->  , regdate timestamp default now()
    ->  , hit int default 0
    -> ) default character set utf8;
